Site one-page para a Kido Despachante, empresa com mais de 30 anos de atuação em Garopaba/SC, credenciada ao DETRAN/SC e associada à ADOTESC.

```

```

## 📦 Deploy

### Netlify (recomendado)
1. Acesse [netlify.com](https://netlify.com)
2. **Add new site → Deploy manually**
3. Arraste a pasta do projeto
4. Pronto ✅

### GitHub Pages
1. Suba o repositório no GitHub
2. Vá em **Settings → Pages**
3. Selecione branch `main` e pasta `/root`
4. Acesse via `https://seu-usuario.github.io/kido-despachante`
